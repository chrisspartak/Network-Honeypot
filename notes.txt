PROJECT GOALS – FAKE FTP HONEYPOT

1. I want to create a Fake FTP Honeypot in Python.
2. I want to capture:
   - attacker login attempts
   - attacker IP addresses
   - usernames and passwords used
   
3. I want to understand:
   - how FTP authentication works
   - how honeypots trick attackers
   - how logging works in Python

4. I want to learn:
   - how to use pyftpdlib
   - how to structure a Python server script
   - how to store logs in a file

5. Optional future goals:
   - fake directory listing
   - fake FTP commands
   - email alert when someone logs in
